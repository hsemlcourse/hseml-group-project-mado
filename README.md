## Описание задачи

**Задача:** Регрессия (предсказание числового значения)

**Что предсказываем:** Среднюю цену одного авокадо (`AveragePrice`) в USD на основе исторических данных о продажах.

**Датасет:** [Avocado Prices](https://www.kaggle.com/datasets/neuromusic/avocado-prices) с Kaggle  
- Источник: Hass Avocado Board  
- Период: 2015–2018 гг.  
- Частота: еженедельные данные  
- Объем: 18 249 строк, 14 колонок 

**Целевая метрика:** 
- **MAE (Mean Absolute Error)** — основная метрика, интерпретируемая в долларах
- **R² (R-squared)** — дополнительная метрика для оценки объясненной дисперсии

## Качество кода (Линтер)

Для проверки качества кода используется `flake8`.

>Для запуска можно воспользоваться Makefile. Для получения списка досутпных команд `make help`
