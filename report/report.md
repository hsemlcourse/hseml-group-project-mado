## 1. Введение и постановка задачи

- **Цель проекта:** разработать модель машинного обучения для предсказания средней цены авокадо (`AveragePrice`) в США.
- **Практическая ценность:**  
Модель может быть полезна производителям, поставщикам и ритейлерам для прогнозирования цен, планирования продаж и оптимизации запасов.
- **Формулировка задачи:** регрессия 
- **Обоснование метрики качества:** Основная метрика **MAE (Mean Absolute Error)** в долларах США.  
Дополнительная метрика — **R²** для оценки объяснённой дисперсии.

---

## 2. Поиск и описание данных

- **Источник данных:** Датасет взят с Kaggle — [Avocado Prices](https://www.kaggle.com/neuromusic/avocado-prices).  
Он содержит исторические данные о продажах авокадо в разных регионах США за 2015–2018 годы.
- **Описание датасета:**
  - Размер: 18 249 строк, 14 столбцов.
  - Целевая переменная: `AveragePrice` (средняя цена одного авокадо).
  - Основные признаки: объёмы продаж по PLU-кодам (4046, 4225, 4770), Total Volume, Total Bags, Small/Large/XLarge Bags, тип авокадо (`type`), регион (`region`), дата.

## 3. Обработка и подготовка данных

- **Полная очистка:**
  - Пропуски: отсутствуют
  - Дубликаты: отсутствуют
  - Выбросы: проведён анализ распределений визуализированием и через межквартильный размах. Применено логарифмирование
  - Типы данных: приведены к корректным (`Date` → datetime)

  - Исходное количество признаков 13
  - После Feature Engineering 31

- **Созданные признаки:**

  - Бинарный признак: `has_XLarge`.
  - One-hot encoding: `type` и `region`.
  - Временные признаки: год, квартал.
- **Визуализации:** 
  - Распределения признаков и целевой переменной.

- **Сплит данных:**
  - Использован (time-series) сплит
  - Train: данные до 31.12.2017 (16 953 строки).
  - Test: данные за 2018 год (1 296 строк).

## 4. Baseline-модель

- **Модель:** DummyRegressor
- **Результаты:** 
  - MAE: 0.2519 USD
  - R²: -0.0423
- **Цель:** точка отсчёта для сравнения с более сложными моделями
